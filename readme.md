#### Restfull Server

This is a simple Movie Finder API built using Node.js and Express that connects to the OMDb API to fetch movie data. It allows users to search movies by title and get detailed information using IMDb ID. The project is structured with separate routes and controllers for clean code organization. It uses Axios for API requests and dotenv for managing environment variables securely. The server runs on a local port and returns JSON responses for all requests.
